## 📚 Course Overview

This repository contains daily study materials, code examples, assignments, and projects for the NAVTTC 3-month intensive AI course. The program focuses on practical, hands-on learning with 80% practical work and 20% theory.

### Course Objectives
- Develop employable AI and Machine Learning skills
- Gain expertise in Natural Language Processing (NLP)
- Prepare for Microsoft Azure AI Associate certification
- Build real-world AI projects and solutions

### Learning Outcomes
By course completion, students will master:
- Core AI and Machine Learning concepts
- State-of-the-art ML techniques
- Exploratory data analysis and visualization
- Model design, evaluation, and deployment
- Industry-standard tools: scikit-learn, pandas, numpy, TensorFlow, PyTorch, Keras
- NLP fundamentals and advanced techniques
- Microsoft Azure AI services
